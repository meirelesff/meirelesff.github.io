Recursos

Para acompanhar o curso, conhecimentos intermediários em programaçãosão necessários. Nesta página, listo algumas referências e recursos úteis para o aprimoramento desses pré-requisitos.

Programação

R

  • R for Data Science, Garrett Grolemund and Hadley Wickham.
    • Livro essencial, cobre o básico até o intermediário do uso de R e do tidyverse aplicados à Ciência de Dados
  • R Cookbook, Winston Chang
    • Livro prático focado na resolução de problemas comuns
  • RStudio cheatsheets
    • Guias práticos para resolução de diferentes problemas com tidyverse
  • Rweekly
    • Newsletter semanal com novidades sobre R

Python

  • Python for Data Analysis, Wes McKinney
    • Em certo sentido, é um livro similar ao R4DS, bom para iniciantes
  • Learn Python
    • Site com diversos tutoriais e textos sobre Python
  • Python cheatsheets
    • Guias práticos, agora para Python
  • Pycoders
    • Uma das mais lidas newsletters sobre Python
